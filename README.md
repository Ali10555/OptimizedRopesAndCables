<h3>Features:</h3>
<ul>
<li>Spring Damping in all dimensions</li>
<li>Easily tweak midpoint weight and position</li>
<li>Line Renderer or actual mesh generation</li>
<li>Wind Effect with many parameters</li>
</ul>
